### 3. 数据流向分析验证

AICA描述的数据流:
```
ChatToolWindowControl
  → 创建 OpenAIClient + ToolDispatcher
  → 注入到 AgentExecutor
  → AgentExecutor.ExecuteAsync()
    → 构建 conversationHistory (含 ToolDefinitions)
    → 调用 _llmClient.StreamChatAsync(messages, tools)
    → OpenAIClient 发送 HTTP 请求，返回流式 LLMChunk
    → AgentExecutor 处理 LLMChunk:

      - ToolCall → 执行工具 → 返回 ToolResult → 转为 ChatMessage → 添加到历史 → 循环
```
