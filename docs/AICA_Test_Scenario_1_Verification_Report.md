### 1.3 Agent执行流程 ✅

**AICA描述的执行流程**:
```
1. 构建系统提示 (SystemPromptBuilder)
2. 循环迭代 (最多25次)
   a) 调用 LLM (StreamChatAsync)
   b) 接收 LLM 响应
   c) 执行工具 (ToolDispatcher.ExecuteAsync)
   d) 将工具结果添加到对话历史
3. 结束条件
   - 达到最大迭代次数 (25)
   - 用户取消
   - 调用 attempt_completion 工具
   - 调用 condense 工具
```

**代码验证**:
- ✅ maxIterations = 25（AgentExecutor.cs:37）

- ✅ 结束条件准确（attempt_completion、condense、取消、最大迭代）
